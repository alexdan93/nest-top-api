## Description

Simple API with Auth, Product, Review and page HH data aggregation

Project was created following [Udemy course](https://www.udemy.com/course/nestjs-backend-typescript-node-js/)

## Installation

```bash
npm install
```

## Running the app
Make sure you have installed docker
```bash
# development
$ docker-compose up -d
$ npm run dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
